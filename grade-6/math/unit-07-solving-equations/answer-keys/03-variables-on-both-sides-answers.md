# Answer Key: Worksheet 3 — Equations with Variables on Both Sides

---

### Part A: Same Variable on Both Sides

**1.** 3y + 2 = y + 10 → 3y - y = 10 - 2 → 2y = 8 → **y = 4**

**2.** 5y - 4 = 2y + 8 → 5y - 2y = 8 + 4 → 3y = 12 → **y = 4**

**3.** 4y + 1 = 2y + 9 → 4y - 2y = 9 - 1 → 2y = 8 → **y = 4**

**4.** 6y - 3 = 4y + 5 → 6y - 4y = 5 + 3 → 2y = 8 → **y = 4**

**5.** 2y + 7 = 5y - 8 → 7 + 8 = 5y - 2y → 15 = 3y → **y = 5**

**6.** 7y - 6 = 3y + 14 → 7y - 3y = 14 + 6 → 4y = 20 → **y = 5**

**7.** y + 12 = 4y - 3 → 12 + 3 = 4y - y → 15 = 3y → **y = 5**

**8.** 8y - 1 = 3y + 24 → 8y - 3y = 24 + 1 → 5y = 25 → **y = 5**

---

### Part B: Two Variables — Solve for y in Terms of x

**9.** 2y = 6x + 4 → **y = 3x + 2**

**10.** 3y + 9 = 6x → 3y = 6x - 9 → **y = 2x - 3**

**11.** 4y - 10 = 5x + 2 → 4y = 5x + 12 → **y = (5x + 12) / 4**

**12.** 2y + 3 = 4x - 1 → 2y = 4x - 4 → **y = 2x - 2**

**13.** 6y - 2x = 3x + 4 → 6y = 3x + 2x + 4 → 6y = 5x + 4 → **y = (5x + 4) / 6**

**14.** y + 3x = 7x - 8 → y = 7x - 3x - 8 → **y = 4x - 8**

**15.** 5y - x = 2x + 12 → 5y = 3x + 12 → **y = (3x + 12) / 5**

**16.** 8y + 2x = 6x + 20 → 8y = 4x + 20 → **y = (4x + 20) / 8** → simplified: **y = (x + 5) / 2**

---

### Part C: Solve for x in Terms of y

**17.** 3x + 2 = y + 8 → 3x = y + 6 → **x = (y + 6) / 3**

**18.** 5x - 4 = 2y + 6 → 5x = 2y + 10 → **x = (2y + 10) / 5**

**19.** 2x + y = 4x - 10 → y + 10 = 4x - 2x → y + 10 = 2x → **x = (y + 10) / 2**

**20.** 7x - 3y = 4x + 9 → 7x - 4x = 3y + 9 → 3x = 3y + 9 → **x = y + 3**
